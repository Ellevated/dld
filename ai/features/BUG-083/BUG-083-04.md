**Parent:** BUG-083 | **Priority:** HIGH | **Effort:** ~4-6h
**Status:** queued

## Summary

Fix 7 issues in hook infrastructure: detached HEAD handling, glob pattern support, hook validation, regex patterns, sync checking, error messaging, and force-with-lease consistency.

## Root Cause

Hooks migrated from Python to Node.js (ADR-004) but supporting scripts/tests were not fully updated. Edge cases (detached HEAD, globstar) not handled.

## Files to Modify

### 1. `.claude/hooks/utils.mjs` ‚Äî `inferSpecFromBranch()`

**Problem:** `git branch --show-current` returns empty on detached HEAD ‚Üí bypasses spec enforcement entirely.

**Fix:** Add fallback chain:
```javascript
// 1. Try branch name first
let branch = execFileSync('git', ['branch', '--show-current'], {...}).trim();

// 2. Fallback: detached HEAD
if (!branch) {
  try {
    branch = execFileSync('git', ['symbolic-ref', '--short', 'HEAD'], {...}).trim();
  } catch {
    // 3. Extract task ID from commit message
    try {
      const commitMsg = execFileSync('git', ['log', '-1', '--pretty=%s'], {...}).trim();
      const match = commitMsg.match(/(FTR|BUG|TECH|ARCH)-\d+/i);
      if (match) branch = match[0];
    } catch {
      return null; // Fail-safe (ADR-004)
    }
  }
}
```

### 2. `.claude/hooks/utils.mjs` ‚Äî `minimatch()`

**Problem:** `*` converted to `[^/]*` but `**` not handled ‚Üí `ai/diary/**/*.md` doesn't match deep paths.

**Fix:** Use placeholder technique:
```javascript
function minimatch(str, pattern) {
  let re = pattern
    .replace(/[.+^${}()|[\]\\]/g, '\\$&')
    .replace(/\*\*/g, '\x00GLOBSTAR\x00')   // Placeholder for **
    .replace(/\*/g, '[^/]*')                 // * = anything except /
    .replace(/\?/g, '[^/]')
    .replace(/\x00GLOBSTAR\x00/g, '.*');     // ** = anything including /
  return new RegExp(`^${re}$`).test(str);
}
```

### 3. `.claude/hooks/utils.mjs` ‚Äî `extractAllowedFiles()`

**Problem:** Current regex misses dashes, indentation, bold, unicode filenames.

**Fix:** More permissive regex:
```javascript
const pathMatch = trimmed.match(/[`*-]*\s*([^\s`*]+(?:\.[a-zA-Z0-9]+)?(?::\d+(?:-\d+)?)?)[`*]*/);
```

### 4. `scripts/smoke-test.sh` ‚Äî REQUIRED_HOOKS

**Problem:** Lists 6 hooks, actual template has 8.

**Fix:** Add `session-end.mjs` and `validate-spec-complete.mjs` to array. Update count in success message.

### 5. `scripts/check-sync.sh` ‚Äî file patterns

**Problem:** Checks `.py` files but hooks migrated to `.mjs`.

**Fix:** Replace `-name "*.py"` with `-name "*.mjs"` in all find commands.

### 6. `.claude/hooks/run-hook.mjs` ‚Äî error message

**Problem:** `‚úì Allowed: hook not found` is contradictory.

**Fix:** Remove stderr output for missing hooks (silent fail-safe per ADR-004).

### 7. `.claude/hooks/pre-bash.mjs` ‚Äî force-with-lease

**Problem:** Redundant runtime `includes('--force-with-lease')` check when regex already has `(?!-with-lease)`.

**Fix:** Remove redundant line 97 check. Trust regex negative lookahead.
