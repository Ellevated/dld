# BUG-083-01: Template Missing Files

**Parent:** BUG-083 | **Priority:** CRITICAL | **Effort:** ~2h
**Status:** queued

## Summary

Template directory missing critical files that users need on Day 0. Creates broken experience for `npx create-dld` users.

## Findings Addressed

| ID | Severity | Title |
|----|----------|-------|
| CR-001 | CRITICAL | Missing ci-status.sh referenced by autopilot worktree-setup |
| CR-002 | CRITICAL | Missing pre-review-check.py in template/scripts/ |
| JR-001 | CRITICAL | No README.md in template directory |
| JR-002/QA-023/CR-004 | CRITICAL | No .gitignore in template |
| UX-002 | CRITICAL | Broken documentation links in getting started table |

## Root Cause

Template grew organically — new scripts/docs were added to root but never synced to template/. No automated check catches missing files.

## Files to Create

### 1. `template/README.md`
Minimal README explaining the project was created with DLD.

```markdown
# {Project Name}

Created with [DLD Framework](https://github.com/Ellevated/dld).

## Getting Started

1. Run `claude` to start Claude Code
2. Say `/bootstrap` to unpack your idea
3. Say `/spark` to create your first feature

## Commands

- `./scripts/setup-mcp.sh` — Configure MCP servers
- `./scripts/smoke-test.sh` — Validate project structure

## Learn More

- [DLD Documentation](https://github.com/Ellevated/dld/tree/main/docs)
- [Quick Start Guide](https://github.com/Ellevated/dld/blob/main/docs/getting-started/quick-start.md)
```
