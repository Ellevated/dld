# Bug: [BUG-084-07] Node.js requirement not documented

**Status:** queued | **Priority:** P1 | **Date:** 2026-02-14
**Parent:** [BUG-084](BUG-084.md)

## Symptom
All hooks require Node.js (migrated from Python to Node.js per memory context), but neither README nor CLAUDE.md mentions Node.js as a prerequisite.

Files checked:
- `template/CLAUDE.md` — line 29-40 (Quick Start) — no Node.js mentioned
- `template/README.md` — lines 1-20 (Getting Started) — no prerequisites section

New users will hit "node: command not found" errors when hooks run, with no guidance.

## Root Cause
Documentation wasn't updated when hooks migrated from Python to Node.js.

## Fix Approach
Add prerequisites section to both files:

**CLAUDE.md** (before Quick Start):
```markdown
## Prerequisites

- Node.js 18+ (for hooks)
- Claude Code CLI
```

**README.md** (before Getting Started):
```markdown
## Prerequisites

- Node.js 18+
```

## Allowed Files
1. `template/CLAUDE.md` — add Prerequisites section before line 29
2. `template/README.md` — add Prerequisites section before line 6
