# Bug: [BUG-084-19] .mcp.json.example not referenced in Quick Start

**Status:** queued | **Priority:** P2 | **Date:** 2026-02-14
**Parent:** [BUG-084](BUG-084.md)

## Symptom
Lines 29-33 of `template/CLAUDE.md` show Quick Start with manual MCP configuration commands:
```bash
claude mcp add context7 -- npx -y @context7/mcp-server
claude mcp add --transport http exa "https://..."
```
But `.mcp.json.example` exists in template root with pre-configured MCP servers. New users don't know about this easier alternative.

## Root Cause
Quick Start documents the manual CLI approach but doesn't mention the example file. Users waste time typing commands when they could copy `.mcp.json.example` to `~/.claude/mcp.json`.

## Fix Approach
Add note after line 33 mentioning `.mcp.json.example` as an alternative:
```markdown
   ```
2. **Alternative:** Copy `.mcp.json.example` to `~/.claude/mcp.json` for pre-configured MCP setup
3. Run `/bootstrap` to unpack your idea
```

## Allowed Files
1. `template/CLAUDE.md` — add note about .mcp.json.example at line 34

- [ ] .mcp.json.example mentioned in Quick Start
- [ ] Users aware of both manual and copy-file approaches
