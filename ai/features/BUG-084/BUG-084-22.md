```yaml

```

And line 147 says "Run `/skill-writer` with this spec as input."

But there's no brief explanation of:

```markdown

2. Applies all changes in "## Proposed Changes" to CLAUDE.md and .claude/rules/

4. Updates diary index.md to mark processed entries as "done"

**Next step:** Run `/skill-writer` with this spec as input.
```

Also update line 164 output format:
