# create-dld

Create a new DLD (Double-Loop Development) project in seconds.

## Usage

```bash
npx create-dld my-project
cd my-project
claude
/bootstrap
```

## What It Does

1. Creates a new directory with your project name
2. Copies the DLD template (`.claude/`, `ai/`, etc.)
3. Initializes a fresh git repository
4. Shows next steps

## Requirements

- Node.js 18+
- Git
- [Claude Code CLI](https://claude.ai/code)
