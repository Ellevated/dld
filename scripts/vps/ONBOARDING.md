# DLD Cloud Server — Quick Start

Сервер: `5.61.91.190`, юзер: `dld`

---

## 1. Подключение

```bash
ssh dld@5.61.91.190 -t "tmux new -A -s work"
```

Одна команда: подключается по SSH и сразу входит в tmux-сессию `work`.
Если сессия уже есть — подхватывает. Если нет — создаёт.

---

Каждая команда делает `cd` в папку проекта и запускает `claude`.

## 3. Tmux — шпаргалка

Все команды tmux начинаются с **Ctrl+B**, затем клавиша.

### Окна (вкладки)

| Действие | Клавиши |
|----------|---------|
| Новое окно | `Ctrl+B` затем `C` |
| Следующее окно | `Ctrl+B` затем `N` |
| Предыдущее окно | `Ctrl+B` затем `P` |
| Перейти к окну 0-9 | `Ctrl+B` затем `0`...`9` |
| Список всех окон | `Ctrl+B` затем `W` |
| Переименовать окно | `Ctrl+B` затем `,` |
| Закрыть окно | `exit` или `Ctrl+D` |

### Панели (разделение экрана)

| Действие | Клавиши |
|----------|---------|
| Разделить горизонтально | `Ctrl+B` затем `"` |
| Разделить вертикально | `Ctrl+B` затем `%` |
| Переключить панель | `Ctrl+B` затем стрелки |
| Закрыть панель | `exit` или `Ctrl+D` |
| Развернуть/свернуть панель | `Ctrl+B` затем `Z` |

### Сессии

| Действие | Команда |
|----------|---------|
| Отключиться (сессия живёт!) | `Ctrl+B` затем `D` |
| Список сессий | `tmux ls` |
| Подключиться к сессии | `tmux attach -t work` |
| Новая сессия | `tmux new -s имя` |
| Убить сессию | `tmux kill-session -t имя` |

**Важно:** `Ctrl+B` затем `D` — отключает тебя, но сессия продолжает работать. Всё запущенное сохраняется. Подключись обратно через `tmux attach`.

## 4. Типичный workflow

```bash
# 1. Подключился к серверу
ssh dld@5.61.91.190 -t "tmux new -A -s work"

# 2. Запустил Claude в проекте
dow          # откроет Dowry + Claude

# 3. Нужен второй проект? Открой новое окно:
#    Ctrl+B затем C
awad         # откроет Awardybot + Claude

# 4. Переключайся между окнами: Ctrl+B затем N/P или 0/1/2

# 5. Закончил — отключись: Ctrl+B затем D
#    Всё продолжит работать на сервере
```

---

## 5. Настрой себе быстрое подключение (один раз)

Попроси свой Claude Code:

> Добавь мне алиас `ccl` в мой shell config, который подключается к серверу `dld@5.61.91.190` через SSH с tmux, с настройкой ServerAliveInterval=30 чтобы SSH не отваливался.

Или вручную добавь в `~/.bashrc` (Linux) или `~/.zshrc` (Mac):

```bash
alias ccl='ssh -o ServerAliveInterval=30 -o ServerAliveCountMax=5 dld@5.61.91.190 -t "tmux new -A -s work"'
```

После этого просто пишешь `ccl` — и ты на сервере в tmux.

---

## 6. Если SSH отвалился

Не страшно. Tmux сохраняет всё. Подключись заново:

```bash
ccl
```

Все окна, процессы, Claude — всё на месте.
